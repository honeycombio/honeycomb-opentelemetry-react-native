# Local Development

## Prerequisites

**Required:**

You must have `npm` and `yarn` installed.

You can install `yarn` with:
```
npm install -g yarn
```

## Formatting

To lint the code:
```
yarn lint
```

To auto-format code:
```
yarn lint --fix
```

## Example app

To run the example app on iOS in a simulator:
```
yarn install
yarn example ios
```

or Android in an emulator:
```
yarn install
yarn example android
```
